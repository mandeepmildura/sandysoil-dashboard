What's different from the current screen

One status banner answers the only question that matters first. "Everything healthy" / "Running now" / "Controller offline" — colour-coded so a glance from the ute window is enough. No need to scan four vitals tiles to add up the answer.

Bottom nav has 4 destinations, not 5+ behind a More button. Home / Zones / Schedule / History. Pressure becomes a card on Home (it's a vital, not a destination). Admin and device-detail screens move to a top-right menu. Tap targets are 56px including label, comfortable with work gloves.

Live elapsed/remaining replaces "—:—". The current zone card shows --:-- when running. Customers want to know how long until it stops.

"Stop all zones" is one tap and red. When something's wrong, this is the most important action. It currently lives behind a small bordered button at the top-right.

Problem state has actionable next steps. "What to check" + "Message Sandy Soil" on the same screen the alert appears. Less context-switching.

Terminology: just "Zones". Not Valves, not Outputs, not Controller. Pick one word and use it everywhere.

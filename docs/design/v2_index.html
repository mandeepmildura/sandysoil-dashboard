Sandy Soil Dashboard v2

Redesign mockups

Static HTML mockups of the proposed redesign. Open each in a browser to see the visual direction. Nothing is wired up — these are review-only. Code changes happen after sign-off.

Design goals

  • "Is anything wrong?" answered in < 1 second. One status banner at the top of every screen.
  • Mobile-first, glove-friendly tap targets. 48px minimum, 56px for primary actions. No "More" menu hiding important pages.
  • One terminology, used everywhere. Just "Zones" — not Valves, not Outputs, not Controller.
  • Dual personas, one app. Customer view (mobile-first) + admin / contractor view (table of all customers).
  • One unified design system. Manrope + Inter, dark green (#0d4d20) primary, no gradients except status banners, no decorative drop-shadows.
